# 鸡蛋菜煎饼

> 前提：鸡蛋+爱吃的配菜，煎饼（建议买小麦煎饼）

## 设计

1. 鸡蛋葱花煎饼：热锅后喷油然后放煎饼一面，然后再换着煎另一面，打个鸡蛋，待定型后撒些葱花再合上另一面，最后切换另一面煎到酥脆。
2. 鸡蛋葱花虾皮煎饼：多放了虾皮。
3. 鸡蛋葱花虾皮五香煎饼：五香粉即之前买的兰州拉面粉，替换盐。
4. 鸡蛋葱花虾皮魔芋丝煎饼：魔芋丝小零食当配菜。

注意：以上均未使用盐，个人喜欢吃清淡且本来就没有盐。
