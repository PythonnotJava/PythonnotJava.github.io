把 AI 从聊天框里释放出来

RemindAI — 开源桌面 AI 助手，为大模型提供完整的工具外壳，让 AI 直接操作文件、执行代码、管理记忆。

📦 下载最新版 📖 配置指南 ⭐ GitHub

🆕 v1.0.4 更新

  • 长对话思考防卡顿优化
  • 软失效过滤优化，避免过时记忆与新记忆同权重混合
  • 多 Agent 并行理解大量文档编排
  • 可控 AgentLoop 模式：思考 → 编写 → 测试 → 验证 循环流水线
  • 技能支持批量导入
  • 代码高亮文本优化
  • 使用 [archify](https://github.com/tt-a1i/archify) 对对话进行流程图总结

✨ 核心特性

🐚

ToolShell

文件沙盒 + Python/Shell/JS 执行器 + RTK 压缩 60-90% token 消耗，AI 直接操作你的项目。

🔌

MCP 协议

stdio / SSE / Streamable HTTP 三传输全覆盖，工具自动发现、动态注入 Agent。

🧠

向量记忆

Qdrant 语义搜索 + SQLite 持久备份，双写架构自动容灾 + 软失效过滤，跨会话记住重要信息。

🤝

多 Agent 协作

指挥部 / 工作者 / 审查员角色体系 + 权限隔离，产出自动路由注入上下文，支持大量文档并行理解。

🎨

多模型原生适配

OpenAI / Anthropic / Gemini 三协议独立实现，流式推理链 + 多模态图文混合。

🧩

可插拔 Capability

Custom → MCP → ToolShell 三级中间件路由，新增能力只需实现接口 + 注册一行。

📦

四层技能系统

L1 元技能 + L2 全局 + L3 项目临时 + L4 自生成规划中，SKILL.md + tools.json 格式，ZIP 一键导入，支持批量导入。

🔍

Web 搜索

Tavily / Brave / 百度智能搜索，会话级一键开关。

🌐

对外 API 服务

内置 HTTP 服务器，三种端点：OpenAI 聚合、Claude Agent、Claude 代理透传。

🌍

在线访问 Agent

通过浏览器远程访问 RemindAI Agent，支持在线会话管理。

🗜️

上下文压缩

RTK 命令输出压缩 + 上下文智能裁剪，减少 60-90% token 消耗。

🔄

可控 AgentLoop

思考 → 编写 → 测试 → 验证 的循环流水线模式，长对话防卡顿优化。

🐱

全局宠物 Agent

像素猫桌面陪伴 + 右键 AI 问答 + TTS 语音 + 商店背包投喂 + 成就系统。

📤

更多

Schedule 计划管理 / 领域专家 / Gemini 文生图 / OCR / 对话导出 / 中英双语 / Material 3 主题

🧠 记忆 ❌ 无或仅上下文 ✅ 向量语义记忆 + SQLite + 软失效过滤
🔌 工具扩展 ⚠️ 有限 ✅ MCP + 技能 + Capability
🤝 多 Agent ⚠️ 多窗口并排 ✅ 真协作 + 自动路由 + 并行文档理解

🚀 快速开始

# 环境要求: Flutter SDK >= 3.12.1
git clone https://github.com/PythonnotJava/RemindAI.git
cd RemindAI

# Windows
flutter build windows --release --tree-shake-icons

# Linux / macOS
flutter build linux --release --tree-shake-icons
flutter build macos --release --tree-shake-icons
Windows 正式支持 · Linux / macOS 自行编译
